## 📜 설치 가이드

### 프로젝트 가져오기

```
git clone https://github.com/boostcampwm-2021/web31-MAFIA31.git
```

### 코드 실행하기

`packages/client/.env`와 `packages/server/config/*.config.json` 설정 후

```
yarn install
yarn start:dev
```

코드 실행후 http://localhost:3000 에서 확인 가능합니다!
